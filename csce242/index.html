Assignments

Assignment 1 - Basic HTML

This assignment introduced basic elements for an html webpage. This included creating a title, header, and body section. Tables, images, bulleted lists, and links were incorporated into the html webpage.

Assignment 2 - Basic CSS

In this Assignment, we used css to style an html webpage. Styles included adding aesthetics to a navigation bar, creating uniform font and text color, and having sections with various background colors.

Assignment 3 - Page Layout

In this assignment, we used flexbox and media query to design a page layout containing different sections, articles, and asides. The page had a horizontal layout in larger resolutions and a vertical layout in smaller resoltions.

Assignment 5 - Recreating CSS

Using css and html, we recreated one of USC's webpages. The purpose of this assignment is to practice using flexbox and media query to allow a webpage to have different fomats in different resolutions.

Assignment 7 - Intro to JavaScript

Practicing basics of JavaScript including the use of buttons, functions, constants, and more.

Assignment 8 - Conditionals

This assignment employs the use of if statements in JavaScript to add more interaction to a website.

Assignment 9 - Loops

This assignment employs the use of JavaScript loops to simulate an animation of a stick person climbing stairs.

Assignment 10 - Arrays

This assignment practices the use of associative arrays in Javascript.
